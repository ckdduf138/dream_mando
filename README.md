# Dream Mando

React + Vite + Tailwind CSS 프로젝트

## 시작하기

### 설치

```bash
yarn install
```

### 개발 서버 실행

```bash
yarn dev
```

### 빌드

```bash
yarn build
```

### 프리뷰

```bash
yarn preview
```

## 프로젝트 구조

```
src/
├── components/     # 재사용 가능한 UI 컴포넌트
├── pages/          # 페이지 컴포넌트
├── layouts/        # 레이아웃 컴포넌트
├── hooks/          # 커스텀 React 훅
├── utils/          # 유틸리티 함수
├── assets/         # 정적 파일
├── App.jsx         # 메인 앱 컴포넌트
├── main.jsx        # 엔트리 포인트
└── index.css       # 글로벌 스타일
```

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- React Router
- ESLint

## 커스텀 색상

- **Primary**: Sky Blue (하늘색)
- **Secondary**: Purple (보라색)
- **Accent**: Orange (주황색)
